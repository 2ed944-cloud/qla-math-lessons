Terminating vs Repeating — Intuition (no prime factors yet)

Idea: A fraction \( \tfrac{a}{b} \) has a terminating decimal if we can write it as a fraction with denominator \(10, 100, 1000, \ldots\). If we can’t make a denominator like \(10^k\), its decimal will repeat.

Examples: \( \tfrac{1}{4} = \tfrac{25}{100} = 0.25 \) (terminates). \( \tfrac{2}{5} = \tfrac{40}{100} = 0.40 \) (terminates). \( \tfrac{1}{6} \) cannot be rewritten with denominator \(10^k\) exactly → \(0.1666\ldots\) (repeats).

Practical test we’ll use today: **Reduce** \( \tfrac{a}{b} \) first; if there exists some \(k\) so that \( b \) divides \(10^k\), the decimal terminates; otherwise it repeats. (Next lesson you’ll see how this links to prime factors of \(b\).)

Terminating or Repeating? (Fractions)

How to Convert a Repeating Decimal to a Fraction

Pure repeat (no non‑repeating part): Let \(x=0.\overline{ab}\). Multiply by \(100\): \(100x=ab.\overline{ab}\). Subtract: \(100x-x=ab\Rightarrow x=\dfrac{ab}{99}\). For one repeating digit \(x=\dfrac{d}{9}\); for two digits \(x=\dfrac{ab}{99}\); for three digits \(x=\dfrac{abc}{999}\).

With an integer part: \(I.\overline{R}=I+\dfrac{R}{99\ldots9}\) (same number of 9s as the length of the repeating block). Then simplify.
