Key Concepts

  • Like terms have the same variable part (same letters and exponents). Example: 3x and −5x are like terms; 3x and 3y are not.
  • To combine like terms, add/subtract the coefficients (the numbers in front).
  • Constants (numbers with no variable) combine with other constants.
  • Watch negative signs. A good habit: rewrite subtraction as adding a negative.
  • Simplify means: write an equivalent expression with the fewest terms.
Rule: ax + bx = (a+b)x Rule: a − b = a + (−b) Rule: constants + constants = constant

Mini Lesson (Worked Examples)

Example A: 3x + 5x
  1. Both terms have the same variable part: x.
  2. Add coefficients: 3 + 5 = 8.
  3. Answer: 8x
Example B: 8 + 3k + 5 − k
  1. Rewrite subtraction: 8 + 3k + 5 + (−k)
  2. Combine like terms: 3k + (−k) = 2k
  3. Combine constants: 8 + 5 = 13
  4. Answer: 2k + 13
Common trap: You cannot combine unlike terms. 4y + 3 stays as 4y + 3 (a variable term and a constant are different “species”).
