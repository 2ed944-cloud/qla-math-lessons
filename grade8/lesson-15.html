Key Concepts

  • A translation is a slide. Every point moves the same amount horizontally and vertically.
  • Rule form: if you move a horizontally and b vertically, then (x, y) → (x + a, y + b).
  • Right/up are positive; left/down are negative.
  • Translations preserve size and shape (lengths, angles, area stay the same).
  • Two translations combine by adding the horizontal changes and adding the vertical changes.
Rule: (x,y)→(x+a,y+b) Right/Up = + Left/Down = −

Worked Examples

Example A: Translate A(−3, 4) right 5 and down 2
  1. x′ = −3 + 5 = 2
  2. y′ = 4 − 2 = 2
  3. So A′(2, 2)
Example B: Rule (x, y) → (x − 6, y + 2)
  1. “−6” means 6 left; “+2” means 2 up.
Tip: Always compute the translation vector (Δx, Δy). If every point uses the same (Δx, Δy), it’s a translation.
