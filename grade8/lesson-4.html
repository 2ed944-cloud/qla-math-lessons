Primer: What is Rounding & Why it Matters

Core idea

Rounding replaces a number with a nearby value that has fewer digits, to simplify communication or computation while keeping the value close enough for the purpose. We choose a rounding unit \(u\) (like 1, 0.1, 10, 100) and use the half‑up rule.

The half‑up rule
  • Highlight the rounding digit (place you are rounding to) and look one digit to the right.
  • If that digit is 0–4: keep the rounding digit; if 5–9: increase by 1.
  • For negatives, “away from 0” at a tie (e.g., \(-2.5\) → \(-3\)).
Minimal checks: to round to unit \(u\), find the nearest multiple of \(u\). On a number line, the cutoffs are at midpoints \(r\pm \tfrac{u}{2}\).
Examples (explain before doing)
\( 246.38 \) to 1 dp → \( 246.4 \) (since the hundredths digit is 8 ≥ 5)
\( 7{,}950 \) to nearest thousand → \( 8{,}000 \) (since 950 ≥ 500)
\( -12.5 \) to integer → \( -13 \) (tie goes away from 0)

Significant Figures

Count s.f. (discover the rules)

  • Ignore leading zeros.
  • Zeros between non‑zeros count.
  • Trailing zeros count only if there’s a decimal point.
Example: \(1200\) has 2 s.f.; \(1200.\) has 4 s.f.; \(0.00340\) has 3 s.f.
  • “All zeros don’t count.” ❌ Only some zeros are ignored.
  • “Decimal point doesn’t matter.” ❌ It affects trailing zeros.

Scientific Notation (Round Mantissa)

Bounds (when a value is rounded)

If \( x \) rounds to \( r \) to the nearest unit \(u\), then the true value \(x\) is in the interval \( [\,r-\tfrac{u}{2},\; r+\tfrac{u}{2}\,) \). The right endpoint is open.
