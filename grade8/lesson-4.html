Lesson Flow & Success Criteria

Learning Outcomes (LOs)

  • I can round integers and decimals using the half‑up rule (including negatives).
  • I can round to a given number of significant figures and in scientific notation.
  • I can interpret bounds (error intervals) for a rounded value.
  • I can choose suitable rounding for Qatar‑based real‑world contexts and justify my choice.
Teacher note. Invite students to self‑assess at the end with the LO checklist (see final slide).

Learning Objectives & Keywords

Learning Objectives

  • Round whole numbers and decimals to a specified place (nearest 10, 100, 0.1, 0.01, …) using the half‑up rule (symmetric for negatives).
  • Round to a given number of significant figures and in scientific notation.
  • Interpret a rounded value’s bounds (interval of possible true values).
  • Choose suitable rounding for estimation in real‑world contexts and justify reasonableness.
Teacher note. Ask: “Where have you seen rounding today?” (money, transport schedules, measurements).

Keywords (tap for definitions)

half‑up rounding place value decimal places (dp) significant figures (s.f.) scientific notation bounds / error interval estimation

We use the school standard half‑up rule: a trailing 5 rounds away from 0 (e.g., \(-1.5\to -2\)).

Place Value & The Rounding Rule

Read this first: Mark the rounding place, then check the digit to its right (0–4 stay, 5–9 go up). For negatives, a 5 still rounds away from 0.

Highlight the rounding digit and the next digit (to decide up/down).

Result:
Rule: check the digit to the right. If 5–9 → increase by 1; if 0–4 → keep. Negatives: still away from 0 at 5.

Examples

\( 246.38 \) to 1 dp → \( 246.4 \)
\( 7{,}950 \) to nearest thousand → \( 8{,}000 \)
\( -12.5 \) to integer → \( -13 \)
Ask: “Which digit did we look at? Why?” Have students annotate with color on mini‑whiteboards.

Significant Figures

Read this first: Leading zeros never count. Zeros between non‑zeros do count. Trailing zeros count only if there is a decimal point.

Count s.f.

Round to n s.f.

Examples: \(1200\) has 2 s.f.; \(1200.\) has 4 s.f.; \(0.00450\) has 3 s.f.

Scientific Notation (Round Mantissa)

Read this first: Keep mantissa in \([1,10)\). If rounding pushes it to \( \ge 10\), divide by 10 and increase the exponent by 1.

Bounds (when a value is rounded)

Read this first: If \( x \) rounds to \( r \) to the nearest unit \(u\), then the true value \(x\) lies in \( [\,r-\tfrac{u}{2},\; r+\tfrac{u}{2}\,) \).
